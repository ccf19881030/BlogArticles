# Windows下打包C++程序
最近在Windows系统下编写了一个基于Qt5.14.2的C++小工具，需要打包，在网上搜索到两个工具：
1、[Enigma Virtual Box](https://enigmaprotector.com/en/downloads.html)
2、[Inno Setup Compiler](https://jrsoftware.org/isinfo.php)
